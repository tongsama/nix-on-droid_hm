### nix-on-droid
nix-on-droidでhome-managerをセットアップするためのnix及びflakeのrepository
sshdのstartup/stopスクリプトのセットアップ付き


### 使い方
.config/へこのnix-on-droidとhome-managerレポジトリををcloneしたあとに、
次の操作でhome-managerをセットアップし、nix-on-droidからhomeのセットアップを行う


### 注意
セットアップ対象となるバージョン
nix-on-droid: xxxx
nixos,nixpkgs: xxxx
home-manager: xxxx

基本的に上記は揃える必要があり、
nix-on-droidがサポートするバージョンに固定化される
